# 机械钟楼 - 齿轮平台游戏

一个使用 Phaser 3 制作的机械钟楼主题平台跳跃游戏。玩家扮演修钟学徒，在巨大齿轮、摆锤、链条、升降平台和发条门之间穿梭，修复停摆的钟楼。

## 原始需求

> 请用 Phaser 制作机械钟楼齿轮平台游戏，玩家操控修钟学徒在巨大齿轮、摆锤、链条、升降平台和发条门之间穿梭，修复停摆的钟楼。玩家能跳跃、攀链、切换齿轮方向、推动小齿轮和短暂停住机关，关卡节奏要体现机械运动规律：齿轮改变平台，摆锤既是危险也是跳跃时机，发条倒计时会重排路线。失败反馈要说明是被齿轮夹住、掉进机井、错过发条窗口还是被摆锤撞落，玩家复活后能立刻理解该怎么改。

## 项目简介

- **游戏引擎**: Phaser 3
- **构建工具**: Vite 5
- **游戏类型**: 平台跳跃 / 解谜
- **主题**: 机械钟楼 / 蒸汽朋克

### 核心特性

- 🎮 **丰富的机械机关**: 旋转齿轮平台、摇摆摆锤、可攀爬链条、移动升降平台、发条门
- 🕹️ **多样化操作**: 二段跳、攀链、推齿轮、切换齿轮方向、时间暂停
- ⏰ **发条倒计时系统**: 每20秒机关重排，增加游戏挑战性
- 💀 **详细失败反馈**: 4种死亡类型，每种都有针对性的改进提示
- ✅ **检查点系统**: 4个检查点，死亡后从最近检查点复活
- 🎨 **程序化美术**: 所有图形通过代码生成，无需额外资源文件

## 启动方式

### 前置要求

- Node.js >= 18.0.0
- npm >= 9.0.0 或 pnpm >= 8.0.0

### Docker 一键启动（推荐）

#### 前置要求

- Docker >= 20.10
- Docker Compose >= 2.0

#### 启动命令

```bash
docker compose up --build
```

如需后台运行：

```bash
docker compose up --build -d
```

查看日志：

```bash
docker compose logs -f
```

停止服务：

```bash
docker compose down
```

访问地址：http://localhost:3000

### 本地开发启动

#### 1. 安装依赖

```bash
npm install
```

或使用 pnpm：

```bash
pnpm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

或使用 pnpm：

```bash
pnpm dev
```

访问地址：http://localhost:3000

#### 3. 生产构建（可选）

```bash
npm run build
```

构建产物将生成在 `dist` 目录。

#### 4. 预览生产构建（可选）

```bash
npm run preview
```

访问地址：http://localhost:4173

## 游戏操作

| 按键 | 功能 |
|------|------|
| A / ← | 向左移动 |
| D / → | 向右移动 |
| W / ↑ | 向上攀爬（链条上） |
| S / ↓ | 向下攀爬（链条上） |
| 空格 | 跳跃（可二段跳） |
| Q | 切换最近齿轮的旋转方向 |
| R | 打开最近的发条门 |
| E | 短暂暂停所有机关（2秒，冷却8秒） |
| H | 显示/隐藏操作说明 |

## 游戏机制

### 机械元素

1. **齿轮平台**
   - 旋转的齿轮可以作为平台站立
   - 齿牙会夹住玩家造成死亡
   - 顶部平面相对安全
   - 按 Q 可切换附近齿轮的旋转方向

2. **摆锤**
   - 两端最高点时速度最慢，是跳跃最佳时机
   - 最低点时速度最快，非常危险
   - 也可以站在摆锤顶部作为移动平台

3. **链条**
   - 接触链条后按 W/S 可上下攀爬
   - 攀爬时按空格可从链条上跳出
   - 链条上不受重力影响

4. **升降平台**
   - 自动在两点间往返移动
   - 站在平台上会随平台一起移动
   - 到达端点后会短暂停留

5. **发条门**
   - 按 R 可开启附近的发条门
   - 绿色窗口：安全通过时间
   - 橙色窗口：即将关闭警告
   - 门关闭时如果玩家在门内会被夹住

6. **可推动齿轮**
   - 可以推动的小齿轮
   - 也可以站在上面
   - 可用来压住开关或垫高

### 发条倒计时系统

- 每20秒触发一次机关重排
- 最后3秒会有橙色/红色警告
- 重排时会随机改变齿轮方向和升降平台起点
- 重排时所有发条门会关闭

### 时间暂停技能

- 按 E 激活，可暂停所有机关2秒
- 冷却时间8秒
- 右下角有冷却进度指示
- 可用来穿越困难的机关组合

### 死亡类型与提示

| 死亡类型 | 图标 | 改进建议 |
|---------|------|---------|
| 齿轮夹住 | ⚙ | 观察齿牙位置，在间隙时通过，或使用时间暂停 |
| 机井坠落 | 🕳 | 注意平台间隙，善用二段跳 |
| 错过发条 | ⏰ | 在绿色窗口期间快速通过 |
| 摆锤撞击 | 🔨 | 等待摆锤到达最高点时跳跃 |

## 关卡目标

从左侧起点出发，穿越所有机关，到达右侧的金色钟楼目标点，完成钟楼修复！

## 技术说明

- 使用 Phaser 3 的 Arcade 物理引擎
- 所有游戏图形使用 Phaser Graphics API 程序化生成
- 响应式缩放，自动适配不同屏幕尺寸
- 像素艺术风格渲染

## 注意事项

- 游戏需要键盘操作，暂不支持触屏
- 建议使用 Chrome / Firefox / Edge 等现代浏览器
- 如果游戏运行不流畅，可以关闭其他占用资源的程序
